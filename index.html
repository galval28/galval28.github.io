    Senior Design

    My Senior design project is the Learning Holiday Light Project. We are designing a system that allows a user to set a design or pattern to be displayed on holiday light strings. For example if the user wants the front of the tree(or other object that lights are on) to be one color and the back to be another color, the system would handle it. One of the big challenges of the project has been designing a calibration aligrithm that detirmines where on the 3 dimenssional object each light on the strand is.

    My Roles I am the team's signal processing specialist and I have also been a guiding force to make sure we are on track with deadlines and goals.

    Skills and knowledge gained Linux, Raspberry Pi, Python, System design through designing functionality of each piece and software portions.

    Big Picute Contribuiton So far I have drawn up most of our team's system diagrams and developed a camera with a raspberry pi that will take pictures for the calibration process.

    Shoebotics

    Project Overview: Shoebotics is a robot that uses an attachemtn on the user's shoes to follow the user and carry items for the user. Shoebotics was worked on at HackISU spring 2017, and used arduino microcontrollers and nrf transcivers. Our origonal idea for this prject was to track the user's movements by accelerometers attached to their shoes but the acelometers were incomatable witht eh transcievers. I'm most proud of how we were able to find solutions even when out project was limited by what sensors we could use. We ened up using a simple push button embedded in the user's shoe. This project won the hackathon's Most Commercialization Potential prizze.

    Conrtribution:I wired the transceivers, helped with robot driver/wiring, and robot(reciever) code. I also worked with my teamates to integrate all the different parts of the system.
